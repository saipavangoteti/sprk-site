# SPRK — Ignite Your Potential

A stunning, modern website for SPRK, a digital services company specializing in ATS resume optimization, AI chatbots, and premium web development.

## 🌟 Features

- **Modern Glass-morphism Design**: Sleek, contemporary UI with glass effects and smooth animations
- **Custom Cursor**: Interactive cursor experience that follows user movement
- **Responsive Layout**: Fully responsive design that works seamlessly across all devices
- **Smooth Animations**: Fluid transitions and micro-interactions throughout
- **Single Page Application**: Multiple pages (Home, About, Services) with smooth navigation
- **Contact Form**: Functional contact form with validation and toast notifications
- **Service Showcase**: Detailed service pages for ATS Resume Optimization, AI Chatbots, and Premium Websites

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern CSS with custom properties, animations, and glass-morphism effects
- **JavaScript**: Vanilla JS for interactivity and form handling
- **Google Fonts**: Bebas Neue and Barlow font families for typography

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- Local web server (optional, for development)

### Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/sprk-site.git
cd sprk-site
```

2. Open `index.html` in your browser or serve it with a local web server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Or simply open index.html in your browser
```
